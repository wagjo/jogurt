= Jogurt

Yet another Blog. Showcase of using Clojure with hip technologies.

== Tech stack

* JVM
* Leiningen for building
* Clojure, Dunaj
* HTTP-Kit
* Embedded nrepl with Cider handler
** Use ssh tunnel to access it remotely
